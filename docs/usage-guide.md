# 使用指南

本项目的核心用法是：**在项目目录启动 opencode，用自然语言向 AI 助手提问，它会自动调用合适的分析工具帮你完成工作。**

```bash
cd stock-analysis
opencode
```

---

## 用法一：提问分析

直接用中文告诉 AI 你想分析什么，它会自动调用对应的工具。

### 示例 1：查看行情走势

```
你：帮我看看贵州茅台最近 60 个交易日的走势
```

AI 会自动：
1. 调用 A 股行情数据技能获取 sh600519 的日线数据
2. 展示价格走势、涨跌幅、最高最低价等关键信息
3. 用通俗的语言描述走势特征

### 示例 2：技术面分析

```
你：分析一下平安银行的技术指标，重点看 MACD 和 RSI
```

AI 会自动：
1. 获取 sz000001 的行情数据
2. 计算 MACD 和 RSI 指标
3. 解读当前的技术信号（金叉/死叉、超买/超卖等）
4. 给出技术面的综合判断

### 示例 3：基本面分析

```
你：帮我分析宁德时代的财务状况，看看它的盈利能力和成长性
```

AI 会自动：
1. 获取 300750 的财务报表数据
2. 计算 ROE、毛利率、净利率、营收增速等指标
3. 展示最近几年的趋势变化
4. 给出财务健康度的评价

### 示例 4：对比选股

```
你：帮我对比一下贵州茅台和五粮液，从行情和财务两个角度
```

AI 会自动：
1. 获取两只股票的行情走势数据
2. 获取财务指标数据
3. 生成对比表格
4. 指出两者的差异和各自优劣势

### 示例 5：综合分析

```
你：我在考虑买入比亚迪，帮我做一个全面分析
```

AI 会自动：
1. 获取行情数据，分析近期走势和位置
2. 计算技术指标，判断当前技术形态
3. 获取财务数据，分析盈利能力和成长性
4. 综合以上信息给出分析摘要

### 示例 6：保存研究笔记

在完成一次分析后：

```
你：把刚才的分析记一下，保存到笔记里
```

AI 会将分析结论整理成结构化笔记，保存到 `invest-notes/` 目录。下次你问"之前分析过比亚迪吗"，AI 会帮你找到之前的笔记。

### 示例 7：输入外部材料

你可以给 AI 提供研报、新闻链接等材料：

```
你：看看这篇文章 https://xxx.com/article ，结合数据分析一下它说的对不对
```

AI 会结合文章内容和实际数据，帮你验证观点。

## 用法二：创建自定义技能

每个投资者都有自己的分析方法和选股策略。你可以让 AI 帮你把这些方法变成可复用的技能。

### 示例 1：创建一个选股策略技能

```
你：我有一个选股策略，帮我做成一个技能：
    1. 筛选 ROE 连续 3 年大于 15% 的公司
    2. 市盈率低于行业平均
    3. 最近一个季度营收同比增长大于 10%
    4. 资产负债率低于 50%
```

AI 会：
1. 和你确认策略细节（比如数据范围、行业划分标准等）
2. 在 `.agents/skills/` 目录下创建新技能
3. 编写 SKILL.md 说明文件和实现脚本
4. 告诉你以后怎么调用这个技能

创建完成后，你以后只需要说：

```
你：用我的选股策略筛一下
```

### 示例 2：创建行业分析模板

```
你：我经常需要分析白酒行业的公司，帮我做一个白酒行业分析技能，
    重点关注品牌溢价能力（毛利率）、渠道库存（应收账款周转率）、
    现金流质量
```

### 示例 3：创建日常盯盘技能

```
你：帮我做一个盯盘技能，每次我说"盯盘"的时候，自动帮我看：
    1. 上证指数、创业板指的实时行情
    2. 我自选股（茅台、宁德时代、比亚迪）的价格和涨跌
    3. 如果某只跌超过 3% 就提醒我
```

### 技能创建的原则

- **技能越具体越好**：不要做"万能分析"技能，而是做"白酒行业估值对比"这样有明确目标的技能
- **描述要准确**：SKILL.md 的 description 字段决定了 AI 什么时候会调用这个技能，要写清楚触发条件
- **数据要可获取**：技能设计时要考虑数据来源的可行性，确保用到的数据可以通过公开接口获取

更多关于技能创建规范，请参考 [自定义技能创建指南](./custom-skill-guide.md)。

## 提问技巧

### 好的提问方式

- "帮我分析贵州茅台最近的技术面和基本面" — 明确了股票和分析维度
- "对比白酒板块的茅台、五粮液和泸州老窖" — 明确了对比对象
- "宁德时代的 ROE 连续几年在什么水平？" — 明确了具体指标

### 可以优化的提问

- "这个股票怎么样？" — 没说哪只股票，AI 会先问你
- "帮我分析" — 太模糊，建议指明分析对象和维度
- "这个能涨吗？" — AI 不会预测涨跌，但可以分析基本面和技术面信号

### AI 能做和不能做的

| 能做 | 不能做 |
|------|--------|
| 获取行情和财务数据 | 预测股价走势 |
| 计算技术指标 | 给出明确买卖建议 |
| 分析财务健康度 | 提供内幕信息 |
| 对比同行业公司 | 保证分析准确性 |
| 帮你建立分析框架 | 替代你做投资决策 |
