# 个股深度分析框架

确保每次个股综合分析都形成完整的投资逻辑链条，不遗漏关键维度。

**这是一个纯框架型技能**，不包含脚本。它指导 AI 按以下维度组织分析，
并调用其他技能获取数据。

## 分析流程（必须按顺序执行）

### 第一步：公司画像（调用 company-profile）

```bash
python .agents/skills/company-profile/scripts/profile.py <代码>
```

从输出中提取并呈现：
- **一句话定位**：这家公司做什么？在行业中扮演什么角色？
- **核心产品/业务**：按产品拆分的营收占比和毛利率
- **地理分布**：境内/境外收入占比（海外占比高则标注汇率风险）
- **最新业绩**：业绩快报/预告的关键数据

### 第二步：竞争优势与增长逻辑（调用 WebSearch）

搜索以下内容（按需组合关键词）：

- `"<公司名> 核心竞争力 护城河 壁垒"`
- `"<公司名> 在建项目 产能规划 <当年>"`
- `"<公司名> 竞争对手 行业格局 市场份额"`
- `"<公司名> 国产替代 自主可控"`（如涉及）
- `"<行业> 行业规模 市场空间 增速"`

从搜索结果中提炼：
- **护城河**：技术壁垒、客户壁垒、品牌壁垒、成本优势、网络效应（有哪些？强度如何？）
- **增长引擎**：当前增长靠什么？第二曲线是什么？转型进展到哪一步？
- **竞争格局**：行业集中度、主要竞争对手、公司的差异化定位
- **市场空间**：所在赛道的市场规模和增速，天花板在哪里

### 第三步：财务验证（调用 financial-data-analysis）

```bash
python .agents/skills/financial-data-analysis/scripts/finance.py <代码> --years 3
```

不只列数据，要回答以下问题：
- 盈利能力在**改善还是恶化**？（ROE/毛利率/净利率的趋势）
- 增长是否**真实可持续**？（营收增长 vs 应收增长、经营现金流 vs 净利润）
- 财务是否**安全**？（负债率、现金储备、偿债能力）
- 运营效率如何？（存货/应收周转天数是否合理？与同行比呢？）

### 第四步：市场环境（调用 market-context + WebSearch）

```bash
# 行业板块表现
python .agents/skills/market-context/scripts/context.py sector --keyword <行业关键词>

# 个股新闻
python .agents/skills/market-context/scripts/context.py news --code <代码>

# 上游商品价格（如涉及资源/制造业）
python .agents/skills/market-context/scripts/context.py commodity --symbols <品种代码>
```

WebSearch 补充：
- `"<行业> 政策 十五五 2026"`（产业政策方向）
- `"<行业> 景气度 供需 <当年>"`（行业周期位置）

### 第五步：技术面位置（调用 technical-indicator）

```bash
python .agents/skills/technical-indicator/scripts/indicator.py <sh/sz代码>
```

不只报数值，要回答：
- 当前价格在什么**位置**？（高位/低位/震荡区间）
- 趋势方向如何？多指标是否**共振**？
- 有无明显的**风险信号**？（超买/超卖、量价背离）

### 第六步：估值判断

综合前五步信息，回答：
- 当前 PE/PB 是多少？市场为什么给这个估值？
- 对标哪个板块的估值更合理？（传统主业 vs 新业务转型）
- 估值是偏高、合理还是偏低？依据是什么？

### 第七步：催化剂与时间表

基于第二步的搜索结果，梳理：
- 未来 6-12 个月有哪些关键催化剂？（产能投产、客户认证、政策出台、业绩拐点）
- 什么时间点需要重点跟踪？

### 第八步：形成结论

输出格式：

```
## 核心观点
一句话总结投资逻辑。

## 支撑逻辑
1. ...
2. ...
3. ...

## 变量因素
可能改变投资逻辑的关键变量。

## 核心风险
1. ...
2. ...
3. ...

## 关键跟踪节点
| 时间 | 事件 | 影响 |
|------|------|------|
| ... | ... | ... |
```

## 维度完整性检查

分析完成后，AI 应自查以下维度是否覆盖（打勾表示已覆盖）：

- [ ] 公司做什么、靠什么赚钱
- [ ] 营收结构（产品/地区拆分）
- [ ] 竞争优势 / 护城河
- [ ] 增长逻辑 / 第二曲线
- [ ] 竞争格局
- [ ] 财务趋势（改善/恶化）
- [ ] 财务质量（现金流 vs 利润）
- [ ] 行业景气度和政策方向
- [ ] 技术面位置和信号
- [ ] 估值水平和合理性
- [ ] 催化剂时间表
- [ ] 核心风险

如有维度未覆盖（因数据不可得等原因），应在分析中明确标注。

## 注意事项

- 此框架适用于**个股深度分析**场景。简单看行情、查财务等不需要走完整流程
- 第二步的 WebSearch 是定性分析的关键环节，不可跳过
- 分析末尾加上：*以上分析基于公开数据，仅供参考，不构成投资建议。*
- 完成后主动问用户是否需要保存为投资笔记
